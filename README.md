# Challenges for the hack.lu 2025 CTF

This repository contains the challenges created for the hack.lu 2025 Capture The Flag (CTF) competition. Each challenge is organized into its own directory, containing all necessary files to set up the VMs needed and instructions to solve it (if you don't manage it on your own).


### Manual VM setup instructions
Windows images used for the challenges:
- Intro: Windows Server 2025 Core 
- Chal1: Windows Server 2025 Core 
- Chal2: 
  - DC: Windows Server 2022 Core 
  - Srv02: Windows Server 2022 Desktop Experience

Minimum VM specs:
- DC: 1 vCPU, 512MB RAM
- Srv02: 1 vCPU, 1GB RAM

If you don't want to set up the VMs manually, here is a download link to the final exported versions:
https://cloud.fluxfingers.net/s/nEdFYYP5XAcwSGw
